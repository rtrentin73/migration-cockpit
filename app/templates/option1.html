Setup Identity

To setup identity, follow these instructions:

  1. Go to the Google Cloud Console
  2. Click the project drop-down and select or create the project that you want to use for this quickstart
  3. Click the hamburger menu and select APIs & Services > Credentials
  4. Click Create credentials > OAuth client ID
  5. Select Web application
  6. Enter a name for the application, specify a redirect URI (enter http://localhost:5000/option1), then click Create
  7. Click OK to dismiss the resulting dialog
  8. Click the download button (Download JSON) to the right of the client ID
  9. Move this file to your working directory and rename it client_secret.json
